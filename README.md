# Share Scripts

A Claude Code skill for publishing scripts from ~/.claude/scripts/ to the HartreeWorks/claude-scripts repository.

## Documentation

See [SKILL.md](./SKILL.md) for complete documentation and usage instructions.

## Installation

```bash
npx skills add HartreeWorks/skill--share-scripts
```

Or install all HartreeWorks skills: `npx skills add HartreeWorks/skills`
